### Code blocks

Code blocks are pretty, this is a `js` block with the
`prism-react-renderer/themes/nightOwl` theme:

```js
const onClick = () => {
  alert('clicked')
}
render(<button onClick={onClick}>Click Me!</button>)
```

And here is a `js react-live` block, try editing the code:

```js react-live
const onClick = () => {
  alert('clicked')
}
render(<button onClick={onClick}>Click Me!</button>)
```
